Column space, null space & rank

A matrix A maps inputs to outputs. The column space is everything it can reach; the null space is everything it crushes to zero. See singular & determinant.

det ≠ 0, so rank = 2, the null space is just {0}, and A is invertible.

The chain is one statement four ways: det = 0  ⇔  rank < 2  ⇔  null space is more than {0}  ⇔  not invertible. See singular.

Key idea

When det = 0 the columns are dependent, so the output space collapses and loses a dimension. The null space is exactly what gets crushed to zero: bigger collapse, bigger null space.
